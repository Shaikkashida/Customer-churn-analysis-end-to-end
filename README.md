Customer Churn Analysis (End-to-End Project)
##🔹 Overview
This project analyzes telecom customer churn to identify key drivers and quantify revenue impact using data analytics techniques.
##🔹 Problem Statement
Customer churn leads to revenue loss. The goal is to identify high-risk customers and recommend strategies to improve retention.
##🔹 Tools Used
Python (Pandas, NumPy)
SQL (MySQL)
Power BI
##🔹 Dataset
Total Customers: 7043
Features: 21
Target Variable: Churn
##🔹 Key Insights
📌 Month-to-month contracts have highest churn (~43%)
📌 New customers churn the most
📌 Fiber optic users show higher churn risk
📌 Electronic check users show high churn
💰 Business Impact
Historical Revenue Lost: $2.86M
Monthly Revenue at Risk: $139K
